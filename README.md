# ShieldED – AI Student Cyber Guardian  

AI-powered phishing detection and cybersecurity awareness companion designed specifically for students.

---

## Problem Statement  

College students are increasingly targeted by phishing emails, scam internship offers, fake scholarship links, and online fraud. Most existing cybersecurity tools either provide technical warnings or block threats silently without educating users. There is a need for a simple, AI-powered solution that detects threats and clearly explains risks while building long-term digital safety awareness.

---

## Solution Overview  

ShieldED is an AI-powered cybersecurity companion that analyzes suspicious emails, messages, and links in real time.

The app:
- Assigns a clear risk score (Low / Medium / High)
- Highlights suspicious elements
- Explains threats in simple, student-friendly language
- Reinforces learning through interactive quizzes
- Encourages digital hygiene habits

ShieldED combines **AI detection + plain-language explanation + awareness building** into one unified platform.
